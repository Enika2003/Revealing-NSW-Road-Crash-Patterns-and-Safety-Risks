# Revealing-NSW-Road-Crash-Patterns-and-Safety-Risks
In this case study, I transformed real-world data into a magazine-style article that presents statistical findings through clear and engaging storytelling for a non-technical audience. By working through the full process of data exploration, model building, and interpretation.

### Dataset : https://opendata.transport.nsw.gov.au/dataset/nsw-crash-data

### Definitions and notes to support road crash data : https://opendata.transport.nsw.gov.au/data/dataset/06f9cf3d-0a9d-4098-b0f0-fa9efbdd3921/resource/ceafc11f-49a9-441d-b0b4-f53019ee00e1/download/data-manual-static-for-open-data-20211020.pdf
